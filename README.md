# 機能一覧
- ユーザー登録・ログイン機能
- ゲストログイン（ワンクリックかんたんログイン）機能
- CRUD 機能（例：記事・タスク・商品など何かしらのリソース）一覧表示・詳細表示・新規作成・編集・削除入力チェック、エラーメッセージ、日本語/英語メッセージ整備
- 検索・絞り込み・ソートキーワード検索、カテゴリ・タグ絞り込み、作成日の昇順/降順
- ページネーションデータ量が増えたときの表示負荷／UXを意識していることのアピールになる
- 画像アップロードなどのファイル処理（1つはあると良い）プロフィール画像、記事サムネイルなど
